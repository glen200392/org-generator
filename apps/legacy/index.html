📖 快速上手指南 (v3.1)

🚀 操作流程

  • 📥 下載 Excel 範本 → 填入 Level、Dept、Name、Title
  • 📁 上傳 Excel → 資料自動載入
  • 🎨 選擇品牌主題 → 預覽即時更新
  • 🔍 滾輪縮放 / 拖曳平移瀏覽預覽
  • ⬇️ 匯出 PNG / PDF / PPT

🔍 縮放與平移

  • 滾輪：在預覽區滾動滑鼠縮放
  • +/− 按鈕：每次 ±15%
  • 「適合」：整圖自動填滿預覽框
  • 「1:1」：還原 100% 原始尺寸
  • 拖曳：按住左鍵拖曳平移畫面
